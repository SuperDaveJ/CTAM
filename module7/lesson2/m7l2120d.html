Key Assumptions Check Steps

The image shows a step graphic, with the following information being displayed upon selecting each step.
  1. Review the current analytic line; write it down. Without the written words, it is easy to assume everyone understands the same thing. The process of writing it down ensures its clarity.

  2. Articulate all ideas, both stated and unstated. Again, writing these down can help ensure they are clearly understood.

  3. Challenge each assumption. Ask questions about its validity.

    • Must it always be true?

    • Under what conditions might it not be true?

    • Could it have been true in the past but no longer be true today?

    • If it turned out to be wrong, how much impact would that have on analytic line?

  4. Refine the list of key assumptions. Delete the key assumptions that do not hold up under questioning. Add new key assumptions that have been identified.

    • The format used in the 2002 Sniper Case example provides a clear way of presenting the assumptions and your assessment of each. As you refine the list of assumptions in the final step, assumptions which are no longer recognized as “must be true” should be removed from the chart.

    • If you are using this analytic method to analyze competing hypotheses, key assumptions for each should be identified and challenged. Through this process, you will need to refine your set of hypotheses—eliminating, changing, or adding hypotheses as appropriate.
