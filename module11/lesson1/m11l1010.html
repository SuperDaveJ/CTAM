Analytic Exercise

This Analytic Exercise is available as an accessible and a non-accessible version.

Select the Accessible Version of the exercise if you are using a screen reader due to visual impairments or you use the computer keyboard instead of the mouse due to motor impairments. This version of the exercise is virtually identical in content to the other, non-accessible version, but will differ in presentation to provide optimal accessibility.

Select the Non-Accessible Version of the exercise if you do not require the accessibility features described above.

NOTE: If you do not have speakers, please access this module in Accessibility mode.
